use cases

radio slicing

5G-EmPOWER allows deploying custom resource allocation schemes within a network slice. The underlying system enforces strict performance isolation between MVNOs and ensures efficient resource utilization across the network in spite of the nondeterministic nature of the wireless medium.

mac offloading

In 5G-EmPOWER MAC functions are virtualized making it possible to offload part of them as Virtual Network Functions. Only higher MAC functions can be deployed as VNF, while the physical layer and the lower MAC are not virtualized.
