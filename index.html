about open 5g-empower

5G-EmPOWER is an open Mobile Network Operating System for Wi-Fi and LTE networks.

5G-EmPOWER supports high-level programming abstractions for fast prototyping of apps and services.

Multi-tenancy allows supporting verticals with orthogonal requirements over the same network.

The 5G-EmPOWER Community is constantly growing with contributors from all over the world.

  • data-plane

    In 5G-EmPOWER everything is treated as Virtual Network Function (including radio access functions). 5G-EmPOWER currently supports WiFi, LTE, and Click-based Virtual Network Functions.

    learn more

  • management-plane

    Complex network services can be implemented by chaining different Virtual Network Functions. A domain specific language allows performing specific packet processing operations on a precise portion of the flowspace.

    learn more

  • sdk

    Data and management planes are exposed to the network service developers trough high-level programming primitives.

    learn more
